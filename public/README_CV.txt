IMPORTANT: CV Download Setup
=============================

To enable the CV download feature:

1. Place your CV PDF file in this folder (public/)
2. Name it exactly: Harshal_Pimpalshende_CV.pdf
3. The download button in the navbar will automatically work

The file should be named:
Harshal_Pimpalshende_CV.pdf

You can export your LaTeX CV to PDF and place it here.
